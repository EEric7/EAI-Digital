# Freelance Portfolio - Blazor

Site vitrine freelance moderne développé avec **Blazor Web App** et **ASP.NET Core (.NET 8)**.

## Aperçu

Ce projet a pour objectif de présenter une activité freelance à travers un site web moderne, responsive et performant.

Le site permet de :

- Présenter des services

- Afficher un portfolio de projets

- Mettre en avant des compétences techniques

- Permettre aux clients de prendre contact facilement

---

## Fonctionnalités

- Interface moderne et responsive

- Navigation fluide avec Blazor

- Présentation des services

- Section portfolio

- Page à propos

- Formulaire de contact

- Architecture propre et maintenable

- Optimisé pour desktop et mobile

## Installation

### Prérequis

- SDK .NET 8

- Visual Studio 2022 / Rider / VS Code

### Cloner le projet

```bash

git clone https://github.com/votre-utilisateur/freelance-portfolio-blazor.git

```

### Accéder au dossier

```bash

cd freelance-portfolio-blazor

```

### Restaurer les dépendances

```bash

dotnet restore

```

### Lancer le projet

```bash

dotnet run

```
